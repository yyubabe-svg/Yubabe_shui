# ISO模板目录

请将ISO管理体系附表模板放置在此目录下，文件名应为：`管理体系附表-设计部分.docx`

## 模板要求

Word文档模板需要包含以下占位符标记（使用双大括号包裹）：

- `{{project_name}}` - 项目名称
- `{{project_code}}` - 项目编号
- `{{client}}` - 业主/委托单位
- `{{location}}` - 项目地点
- `{{design_stage}}` - 设计阶段
- `{{report_date}}` - 报告日期
- `{{project_manager}}` - 项目经理
- `{{chief_engineer}}` - 总工程师
- `{{quality_level}}` - 质量控制等级
- `{{department}}` - 设计部门
- `{{year}}` - 年份
- `{{month}}` - 月份
- `{{day}}` - 日期（日）
